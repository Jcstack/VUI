# 面包屑 Breadcrumbs

由`v-breadcrumb`修饰，一种简洁的站点路径列表容器 .

```html
<nav class="v-breadcrumb">
  <ul>
    <li><a href="#">Vui</a></li>
    <li><a href="#">
        <span class="has-icon is-small">
          <i class="fa fa-github"></i>
        </span>
      Components
    </a></li>
    <li class="is-active"><a href="#" aria-current="page">Breadcrumb</a></li>
  </ul>
</nav>
```
