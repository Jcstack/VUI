# Form Misc

其他相关的form原生元素 .

### 复选框 checkbox

```html
<label class="v-checkbox">
  <input type="checkbox">
  Remember me
</label>
```

<div class="demo-box has-pad-sm">
  <div class="v-field">
    <label class="v-checkbox">
      <input type="checkbox">
      Remember me
    </label>
  </div>
  <div class="v-field">
    <label class="v-checkbox" disabled>
      <input type="checkbox" disabled>
      Save my preferences
    </label>
  </div>
</div>

### 单选框 radio

```html
<div class="v-control">
  <label class="v-radio">
    <input type="radio" name="os">
    OS X
  </label>
  <label class="v-radio">
    <input type="radio" name="os">
    Linux
  </label>
  <label class="v-radio" disabled>
    <input type="radio" name="os" disabled>
    Windows
  </label>
</div>
```
